## 新技术点
- rem自适应方案
- vue-loader的一些新特性
- scss
- 组件化思想：对一些公共组件和样式进行了抽象,代码复用（组件复用）
- vue-awesome-swiper
- px2rem-loader

## 目录：

	- app
	-- css
	--- element.scss
	--- layout.scss
	--- reset.scss
	-- js
	--- core 核心组件（页面中经常用到的一些组件btn、slider、panel）
	--- earn 挣钱页组件文件夹
	--- home 首页组件文件夹
	--- public 公共组件：页头和页脚
	--- router 路由导航
	--- vip VIP会员页组件文件夹
	--- App.vue 顶级的组件
	--- main.js 入口文件
	-- view
	--- index.html 入口文件
	-- viewport.js 自适应布局方案的js
	-package.json 项目介绍和依赖
	-package-lock.json

## 安装

- npm install
- npm start
